Na aula de hoje vamos aprender a integrar as ferramentas de Git e GitHub para
 poder trabalhar em projetos de forma mais facil e dinamica.

Hoje sera a primeira aula onde conectaremos o repositorio online (que esta no GitHub) 
com o repositorio  da minha maquina.

Ediçao realizada pelo GitHub.


Comandos aprendidods na aula de hoje:
git push -> serve para enviar os arquivos da maquina 
        local para o repositorio online no GitHub
git pull -> serve para atualizar o repositorio local
        com os dados do repositorio online no GitHub
